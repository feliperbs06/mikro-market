# mikro-market

## Language

All implementation is in English — code, identifiers, comments, commit messages, docs, and issues. Only user-facing UI copy is in pt-BR.

## Agent skills

### Issue tracker

Issues live in this repo's GitHub Issues (`gh` CLI); external PRs are not a triage surface. See `docs/agents/issue-tracker.md`.

### Triage labels

Canonical label vocabulary (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`). See `docs/agents/triage-labels.md`.

### Domain docs

Single-context (`CONTEXT.md` + `docs/adr/` at the repo root). See `docs/agents/domain.md`.
